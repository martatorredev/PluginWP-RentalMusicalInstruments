Author: Marta Torre
Tags: woocommerce
Requires PHP: 5.6
Requires at least: 4.4
Tested up to: 5.5

Rent Musical Instruments with WooCommerce website.

== Installation ==

1. Upload the entire 'Woocommerce Alquiler Instrumentos' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
